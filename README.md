# Reinforcement Learning Graduate Student Environment(강화학습 대학원생 에이전트)

This simple reinforcement learning (RL) environment file can serve as a tutorial for RL programming using python.
이 강화학습 가상환경을 통해 기본적인 강화학습 프로그래밍 구조와 각 메소드의 역할을 쉽게 파악할 수 있습니다.

In this RL environment for Gymnasium, RL Agent can move around home, lab, and karaoke. It will learn how to behave well in the gradaute school, without getting expelled or having mental burnout. 
해당 강화학습 환경에서 대학원생 에이전트는 집, 연구실, 노래방을 오갈 수 있습니다. 퇴학 당하거나 번아웃 오지 않고 잘 졸업하게 학습시켜 봅시다.

Although there is no English tutorial yet, you can machine translate the Korean one below. I'll write the English one if demand arises.
한국어 튜토리얼 및 설치 가이드는 이 링크를 참고하시길 바랍니다.
https://docs.google.com/document/d/1oRslebHK2009GHblLYoQJBYeagG6hvdikvR-RiD7oxc/edit?usp=sharing
